# Portfolio de Manon LOCCA

Ce portfolio est un projet vivant que j'enrichis au fil de ma formation. Il regroupe les compétences acquises lors de mon cursus ainsi que les technologies explorées en autodidacte sur mon temps libre.

## Technologies utilisées
* **HTML5 / CSS3** : Structure et design responsive.
* **JavaScript** : Interactivité.

## Évolutions à venir
Il est voué à évoluer constamment. Mon prochain objectif est d'y **implémenter React** afin de mettre en pratique mes derniers modules de cours et d'améliorer l'expérience utilisateur.
