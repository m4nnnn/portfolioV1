# Personal Portfolio V1 - m4nnnn

This is the first version of my personal portfolio. It serves as a central hub to showcase my journey as a developer, featuring my projects, technical skills, and contact information.

## Overview

The goal of this project was to create a professional and clean platform to present my work. It reflects my growth from basic HTML/CSS to more advanced styling techniques and responsive design.

## Key Features

- **Project Showcase:** A curated list of my best work (Frontend Mentor challenges, Java tools, etc.).
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **SASS Architecture:** Built using a modular SCSS structure for clean and maintainable code.
- **Modern UI/UX:** Focused on typography, spacing, and subtle interactive elements.

## Built with

- **HTML5** (Semantic structure)
- **CSS** (Modular styling with variables and mixins)
- **Flexbox & CSS Grid** (Layout management)
- **JavaScript** (For interactive components and navigation)

## What I learned from this project

### 1. Building a Personal Brand
This project taught me how to choose a consistent color palette and typography that represents my professional identity.

## 2. Deployment & Optimization
I focused on optimizing image sizes and assets to ensure fast loading speeds for a better user experience.
